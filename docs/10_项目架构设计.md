# 10 项目架构设计

## 项目目标

本项目希望实现一个完全本地部署的 AI 问答系统。

最终支持：

- 本地部署
- CPU 推理
- OpenAI Compatible API
- RAG
- PDF 知识库
- Web 聊天页面

---

## 当前架构

目前已经完成：

```

↓

↓

↓

```
