# 01 环境配置

## 项目简介

本项目基于 llama.cpp 在 WSL2 Ubuntu 环境下部署 Qwen2-7B-Instruct GGUF，实现本地 CPU 推理，并逐步扩展为 RAG 问答系统。

## 开发环境

| 软件 | 版本 |
|------|------|
| Windows | Windows 11 |
| WSL | WSL2 |
| Ubuntu | Ubuntu 24.04 LTS（根据实际修改） |
| GCC | gcc |
| CMake | cmake |
| Git | git |

## 安装依赖

```bash
sudo apt update

sudo apt install -y \
build-essential \
cmake \
git
```

查看版本：

```bash
gcc --version
g++ --version
cmake --version
git --version
```
